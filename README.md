# Laptop Price Prediction — Regression Project

## Overview

This project predicts laptop prices based on product specifications using regression-based machine learning models.

## Problem Statement

Laptop prices depend on multiple specifications such as brand, processor, RAM, storage, GPU, screen size, and operating system. This project analyzes these factors and builds a model to estimate laptop prices.

## Project Workflow

- Cleaned and prepared laptop specification data
- Performed exploratory data analysis to understand price patterns
- Encoded categorical features such as brand, processor, GPU, and operating system
- Trained and compared regression models
- Evaluated model performance using regression metrics

## Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Evaluation Metrics

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Technologies & Skills

Python, Pandas, NumPy, Scikit-learn, Regression, Data Analysis, Exploratory Data Analysis
